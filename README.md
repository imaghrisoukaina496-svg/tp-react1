# TP React - Composants et Props

## 👩‍💻 Auteure
- **Nom :** Soukaina Imaghri

## 📋 Description
Projet React réalisé dans le cadre d'un TP sur les composants 
fonctionnels et les props.

## 🚀 Lancement
npm install
npm start
http://localhost:3000

## 🧩 Composants créés

| Composant      | Props                        |
|----------------|------------------------------|
| HelloWorld     | Aucune                       |
| Greeting       | prenom                       |
| Profil         | utilisateur, taille          |
| Voiture        | marque, modele, couleur      |
| ListeCourses   | elements                     |

## 📚 Concepts appris
- Composants fonctionnels React
- Utilisation des props
- Validation avec PropTypes
- defaultProps
- Rendu de listes avec .map()

## ✅ Résultat
Application fonctionnelle affichant :
- Message Hello World avec infos personnelles
- Salutation dynamique
- Profil avec photo
- Liste de voitures
- Listes de courses dynamiques
